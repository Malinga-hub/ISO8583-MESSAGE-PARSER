# ISO 8583:1993 Message Parser 
### java implementation using springboot 3

# TECH STACK
Brief description of tech stack used

* [Framework: Springboot 3](https://start.spring.io/)
* [Language: Java 21 (LTS)](https://www.oracle.com/java/technologies/downloads/#java21)
* [Frontend: Thymeleaf, Javascript, Bootstrap css](https://www.thymeleaf.org/doc/tutorials/2.1/usingthymeleaf.html#introducing-thymeleaf)
* [Backend: Springboot 3.4.0](https://start.spring.io/)
* [Database: Mysql 8](https://dev.mysql.com/downloads/mysql/8.0.html)
* [Authentication: Auth0](https://auth0.com/)
* [Authorization: custom ROLE_PRIVILEGES implementation](https://www.baeldung.com/role-and-privilege-for-spring-security-registration)
* [Theme: Tabler (free open source dashboard)](https://tabler.io/docs/getting-started/download)
